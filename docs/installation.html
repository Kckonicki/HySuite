Installation Guide

Get HySuite up and running on your Hytale server in minutes.

Requirements

  • Hytale Server - Latest version
  • Java 17 or higher
  • No external dependencies - HySuite is all-in-one!

Quick Installation

Step 1: Download

Download the latest release from the download page or GitHub releases.

Step 2: Install

Copy the JAR file to your server's plugins folder:

copy hysuite-1.0.0.jar %HYTALE_HOME%/Server/plugins/

Step 3: Start Server

Start your Hytale server. HySuite will load automatically and create default configuration files.

Success! You should see [HySuite] HySuite loaded with rank system! in your server logs.

First Time Setup

Enable the Plugin

If the plugin doesn't load, check your server config:

# Server/config.json
"Mods": {
  "com.Buddytohelpu:HySuite": {
    "Enabled": true  // Make sure this is true
  }
}

Created Files

On first run, HySuite creates:

Server/data/plugins/HySuite/
├── config.json          # Main configuration
├── ranks.json           # Rank definitions (2 defaults)
├── homes.json           # Player homes (empty)
├── warps.json           # Server warps (empty)
└── lang/
    └── en.json          # English language file

Verify Installation

Check Server Logs

Look for these messages in your server log:

[INFO] [PluginManager] - com.Buddytohelpu:HySuite from path hysuite-1.0.0.jar
[INFO] [HySuite|P] No existing config found, will create fresh config.
[INFO] [HySuite|P] [EN] Created language file with 149 keys
[INFO] [HySuite|P] Saved 2 ranks to ranks.json
[INFO] [HySuite|P] HySuite loaded with rank system!
[INFO] [PluginManager] Enabled plugin com.Buddytohelpu:HySuite

Test Commands

Try these commands in-game or console:

/hysuite        # Show main command
/hys rank       # Open rank GUI (requires permissions)
/homes          # Open home GUI
/warps          # Open warp GUI

Initial Configuration

Set Up Permissions

Give yourself admin permissions (using LuckPerms or similar):

/lp user <name> permission set hysuite.admin.ranks true
/lp user <name> permission set hysuite.fly true
/lp user <name> permission set hysuite.vanish true

Set Spawn Point

/setspawn

Create First Warp

/setwarp spawn

Troubleshooting

Plugin Not Loading

Check:
  • JAR is in correct plugins folder
  • Java 17+ is installed
  • Plugin is enabled in config.json
  • Server logs for error messages

Commands Not Working

  • Verify permissions are set correctly
  • Check command cooldowns aren't active
  • Ensure player isn't in warmup period

GUI Not Opening

  • Verify UI files are in JAR
  • Check server log for UI-related errors
  • Try: /hys rank instead of direct GUI command
