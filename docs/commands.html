Commands Reference

Complete list of all HySuite commands with usage examples and descriptions.

Tip: Most commands have configurable cooldowns and warmups. Check the configuration guide for details.

Main Commands

Command Description Permission
/hysuite Main admin command with subcommands Various (see admin section)
/hys Short alias for /hysuite Various (see admin section)

Teleport Commands (TPA)

Command Description Usage
/tpa <player> Request to teleport to another player /tpa Buddytohelpu
/tpahere <player> Request a player to teleport to you /tpahere Buddytohelpu
/tpaccept Accept a pending TPA request /tpaccept
/tpdeny Deny a pending TPA request /tpdeny
/tpcancel Cancel your outgoing TPA request /tpcancel

TPA Example Workflow

# Player1 wants to visit Player2
/tpa Player2

# Player2 sees notification and accepts
/tpaccept

# Player1 teleports to Player2 after warmup
# [5 second warmup, then teleport]

Home Commands

Command Description Usage
/sethome [name] Set a home at your current location /sethome base
/home [name] List your homes or teleport to a specific home /home or /home base
/delhome <name> Delete a saved home /delhome old_base
/homes Open home management GUI /homes

Home System Features

  • Default limit: 3 homes per player
  • VIP limit: 10 homes (configurable)
  • Beautiful GUI for management
  • Configurable cooldowns
  • Warmup delays for teleportation

Home management GUI with quick teleport and delete options

Warp Commands

Command Description Permission
/warp [name] List server warps or teleport to a specific warp None (public access)
/warps Open warp list GUI None
/setwarp <name> Create a warp at current location hysuite.setwarp
/delwarp <name> Delete a server warp hysuite.delwarp

Warp Examples

# Admin creates warps
/setwarp spawn
/setwarp shop
/setwarp pvp

# Players use warps
/warp spawn
/warps  # Opens GUI

Cooldowns & Warmups

Understanding Delays:
  • Warmup - Delay before teleport happens (can be cancelled by movement)
  • Cooldown - Delay before command can be used again

Default Timings

Command Warmup Cooldown
TPA 5 seconds 30 seconds
Home 5 seconds 30 seconds
Warp 5 seconds 30 seconds
Spawn 5 seconds None

Bypass: Use hysuite.cooldown.bypass permission to skip all delays.

Color Support

HySuite supports multiple color formats in messages:

MiniMessage Tags

<red>Error message</red>
<green>Success!</green>
<#FF5555>Custom hex color</>
<bold>Bold text</bold>
<italic>Italic text</italic>

Legacy Codes

&cError message
&aSuccess!
&#FF5555Custom hex color
&lBold &oItalic

Named Colors

Supported color names: BLACK, DARK_BLUE, DARK_GREEN, DARK_AQUA, DARK_RED, DARK_PURPLE, GOLD, GRAY, DARK_GRAY, BLUE, GREEN, AQUA, RED, LIGHT_PURPLE, YELLOW, WHITE, ORANGE, PURPLE, PINK, LIME, CYAN

Pro Tips

Efficiency Tips:
  • Use /homes GUI for quick home management
  • Use /warps GUI to see all available warps
  • Use /reply instead of typing player names
  • Set multiple homes for different purposes (base, farm, mine)
  • Admin: Use rank GUI instead of commands for bulk operations
