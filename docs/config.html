Configuration Guide

Customize HySuite to match your server's needs with these configuration options.

Important: After editing configuration files, use /hys reload to apply changes without restarting the server.

File Locations

All configuration files are located in:

Server/data/plugins/HySuite/
├── config.json          # Main configuration
├── ranks.json           # Rank definitions
├── homes.json           # Player homes (auto-managed)
├── warps.json           # Server warps (auto-managed)
└── lang/
    ├── en.json          # English language
    └── fr.json          # French language

Main Configuration (config.json)

Default Configuration

{
  "language": "en",
  "teleportWarmup": 5,
  "teleportCooldown": 30,
  "maxHomes": 3,
  "vipMaxHomes": 10,
  "rtpMinDistance": 100,
  "rtpMaxDistance": 5000,
  "vipCooldownReduction": 0.5,
  "enableRankSystem": true,
  "enableJoinMessages": true
}

Configuration Options

Option Type Default Description
language String "en" Language file to use (en, fr)
teleportWarmup Integer 5 Seconds before teleport (can be cancelled)
teleportCooldown Integer 30 Seconds before command can be used again
maxHomes Integer 3 Maximum homes for default players
vipMaxHomes Integer 10 Maximum homes for VIP players
rtpMinDistance Integer 100 Minimum blocks from origin for /rtp
rtpMaxDistance Integer 5000 Maximum blocks from origin for /rtp
vipCooldownReduction Float 0.5 VIP cooldown multiplier (0.5 = 50% faster)
enableRankSystem Boolean true Enable/disable rank system
enableJoinMessages Boolean true Enable/disable custom join messages

Rank Structure

Field Type Description
id String Unique rank identifier (lowercase, no spaces)
displayName String Human-readable name shown in GUIs
priority Integer Rank weight (higher = more important)
permissions Array List of HySuite permissions

Language Files

Supported Languages

  • English - lang/en.json (149 keys)
  • French - lang/fr.json (149 keys)

Changing Language

Edit config.json:

{
  "language": "fr"  // Change to French
}

Then reload: /hys reload

Language File Structure

{
  "prefix": "[HySuite]",
  "teleport": {
    "warmup": "Teleporting in {seconds} seconds...",
    "cancelled": "Teleport cancelled!",
    "success": "Teleported successfully!"
  },
  "home": {
    "set": "Home '{name}' has been set!",
    "deleted": "Home '{name}' has been deleted!",
    "notFound": "Home '{name}' not found!"
  }
}

Creating Custom Language

  1. Copy lang/en.json to lang/custom.json
  2. Translate all values (keep keys the same)
  3. Set "language": "custom" in config.json
  4. Reload: /hys reload

Performance Tuning

High-Performance Settings

For servers with many players:

{
  "teleportWarmup": 3,
  "teleportCooldown": 60,
  "maxHomes": 5,
  "vipMaxHomes": 15,
  "rtpMinDistance": 200,
  "rtpMaxDistance": 3000
}

Optimization Tips

  • Higher cooldowns reduce server load
  • Lower max homes reduces database size
  • Smaller RTP range = faster calculations
  • Disable unused features

Backup & Restore

Manual Backup

# Stop server or use /hys reload first

# Backup all data
cp -r Server/data/plugins/HySuite/ backup/HySuite-backup/

# Backup specific file
cp Server/data/plugins/HySuite/homes.json backup/

Restore from Backup

# Stop server

# Restore all data
cp -r backup/HySuite-backup/ Server/data/plugins/HySuite/

# Restore specific file
cp backup/homes.json Server/data/plugins/HySuite/

# Start server

Troubleshooting Configuration

Config Not Loading

  1. Check JSON syntax (use jsonlint.com)
  2. Verify file permissions (readable by server)
  3. Check server log for error messages
  4. Try deleting config.json (will regenerate defaults)

Changes Not Applying

  1. Use /hys reload after editing
  2. Verify you edited correct file location
  3. Check for typos in option names
  4. Restart server if reload doesn't work

Invalid JSON Error

Common mistakes:

  • Missing comma between entries
  • Extra comma after last entry
  • Missing quotes around strings
  • Incorrect bracket nesting

  • Validate JSON before applying
  • Start with defaults, adjust gradually
